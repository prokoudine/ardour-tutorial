Impariamo come installare Ardour sul tuo sistema operativo.

## Windows

Scarica il programma di installazione ed eseguilo come al solito. Su Windows 11, potrebbe apparire un avviso di Microsoft Defender che indica che il programma di installazione è un file non comune. [Questa guida](https://ardour.org/windows_defender.html) spiega come risolvere il problema.
## macOS

Scarica il file `.dmg` e fai doppio clic su di esso. Quando si apre la finestra, trascina l'icona di Ardour e rilasciala sull'icona Applicazioni per installarlo.
## Linux

1. Scarica il file `.run`
2. Vai alla cartella in cui è stato salvato il file:
```bash
$ cd /cartella/dove/hai/salvato/il/file
```
3. Esegui il programma di installazione nel terminale:

```bash
/bin/sh ./<NOME_DEL_FILE_SCARICATO>.run
```

4. Segui le istruzioni.

**Continua**

Avviamo Ardour adesso.
