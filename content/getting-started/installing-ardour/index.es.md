Aprendamos cómo instalar Ardour en tu sistema operativo.

## Windows

Descarga el instalador y ejecútalo como de costumbre. En Windows 11, es posible que veas una advertencia de Microsoft Defender indicando que el instalador es un archivo poco común. [Esta guía](https://ardour.org/windows_defender.html) explica cómo proceder en ese caso.

## macOS

Descarga el archivo `.dmg` y haz doble clic sobre él. Cuando se abra la ventana, arrastra el icono de Ardour y suéltalo sobre el icono de Aplicaciones para instalarlo.

## Linux

1. Descarga el archivo `.run`.
2. Navega hasta la carpeta donde se haya guardado:

```bash
$ cd /carpeta/donde/guardaste/el/archivo
```

3. Ejecuta el instalador en la terminal:

```bash
/bin/sh ./<NOMBRE_DEL_ARCHIVO_DESCARGADO>.run
```

4. Sigue las instrucciones.

**Continuación**

Ahora vamos a iniciar Ardour.
